# E-Commerce Course App

A full-stack e-commerce course platform built using React, Node.js, Express, MySQL, and Razorpay integration for secure online payments.

The application allows users to browse courses/products, search and filter items, add products to cart or wishlist, and purchase multiple products in a single order.


# Features

- Browse courses/products
- Search functionality
- Product filtering
- Add to Cart
- Wishlist management
- Buy multiple products at once
- Razorpay payment integration
- REST API backend
- Responsive frontend UI
- MySQL database integration
- Product/course management
